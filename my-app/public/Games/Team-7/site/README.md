# Bærekraft og Natur - Nettside

En fin nettside om bærekraft, miljø og natur.

## Hvordan kjøre nettsiden

### Metode 1: Åpne direkte i nettleser
Dobbelklikk på `index.html` for å åpne den direkte i nettleseren din.

### Metode 2: Bruk lokal server (anbefalt)
1. Åpne en terminal/kommandolinje i denne mappen
2. Kjør en av følgende kommandoer:

**Hvis du har Python installert:**
```bash
python server.py
```

**Eller hvis du har Node.js installert:**
```bash
npx http-server -p 8000
```

**Eller bruk batch-filen (Windows):**
Dobbelklikk på `start-server.bat`

3. Åpne nettleseren og gå til: `http://localhost:8000`

### Metode 3: Bruk VS Code Live Server extension
1. Installer "Live Server" extension i VS Code
2. Høyreklikk på `index.html`
3. Velg "Open with Live Server"

## Filer
- `index.html` - Hovedfilen med nettsidens struktur
- `style.css` - Styling og design
- `script.js` - Interaktivitet og animasjoner
- `server.py` - Enkel Python HTTP server (valgfri)
