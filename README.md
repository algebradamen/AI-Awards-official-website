# AI Awards Official Website

This is the official repository for Tangen High School's AI Awards website.

## Getting Started

To start the application, follow these steps:

1. Navigate to the `my-app` directory:
   ```bash
   cd my-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`

## Team-5 (Aftertaste) Installation

Team-5 has a separate Vite-based React application. To run it:

1. Navigate to the Team-5 folder:
   ```bash
   cd my-app/public/Games/Team-5
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

**Important Notes:**
- You MUST be in the Team-5 (Aftertaste) folder to run it
- Please wait 5-8 seconds on every scene (don't scroll fast)
- On Awards Day, let Team 5 handle the scrolling
